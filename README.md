# voidapp

Приложение VOID разработали Котиев А. и Голяницкий Я. (группа 11-ВТИ23). Это минималистичное социальное приложение для публикации постов с текстом и изображениями. Есть главная лента, экран создания поста, профиль пользователя и настройки. Пользователи могут ставить лайки, комментировать посты и управлять темой, приватностью и видимостью реакций. https://wired-wired-70693516.figma.site
